# API Test Automation Engine (Postman & Newman)

![API Quality Gate](https://github.com/maccana/api-qa-automation-gate/actions/workflows/api-tests.yml/badge.svg)

A robust, headless test automation framework designed to validate API reliability, security, and performance. This repository demonstrates production-grade API quality assurance by executing comprehensive integration and regression test suites entirely through CLI automation, culminating in detailed, stakeholder-ready visual quality reports.
